# TTGOTS-Dof9-System

使用ArduinoIDE編譯的TTGOTS開發版,版上編號是2.0,TftLcd螢幕是1.8寸,本專案是用於九軸感測與藍芽傳輸與螢幕顯示之整合開源專案。

## 關於功能部分

1.藍牙SPP配對後,會不間斷傳送9軸原始藍芽訊號給連接的設備

2.按鈕A是關機鍵，按鈕C是藍芽配對鍵+關機時啟動重開鍵，按鍵B暫無作用

4.TFT-LCD作為藍芽配對資訊或指令接收顯示使用,不另外顯示9軸資訊
